## Create New TanStack Router App

Start by creating a new TanStack Router app using `create-tsrouter-app`. Use `--interactive` and `--framework Solid` parameters to create a new Solid.js project. Select **Solid-UI** and **Start** template when prompted:

<NpmCommand type="create" command="tsrouter-app@latest --interactive --framework Solid" />

```bash showLineNumbers=false
◆  What add-ons would you like for your project?
│  ◻ Form
│  ◻ Sentry
│  ◼ Solid-UI (Add Solid-UI to your application.)
│  ◼ Start (Add TanStack Start for SSR, API endpoints, and more.)
│  ◻ Store
│  ◻ T3Env
│  ◻ TanStack Query
└
```

## Create A `components.json` File

```json
// components.json
{
  "$schema": "https://ui.shadcn.com/schema.json",
  "tailwind": {
    "css": "src/styles.css",
    "config": "",
    "baseColor": "slate",
    "cssVariables": true
  },
  "aliases": {
    "utils": "~/lib/utils",
    "components": "~/components"
  },
  "style": "default",
  "rsc": false,
  "tsx": true
}
```

Change the Tailwind CSS file path and aliases according to your project structure.
