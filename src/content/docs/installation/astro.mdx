### Create New Astro App

Start by creating a new Astro app using `create-astro`:

<NpmCommand type="create" command="astro@latest" />

Once you have created the app, navigate into the project:

```bash  showLineNumbers=false
cd {project-name}
```

Then add Tailwind CSS and Solid.js to your Astro app using Astro CLI:

<NpmCommand type="run" command="astro add tailwind" />

<NpmCommand type="run" command="astro add solid" />

### Add Dependencies

<NpmCommand type="install" command="class-variance-authority clsx tailwind-merge lucide-solid tw-animate-css @kobalte/tailwindcss" />

### Configure Path Aliases

```json ins={6-9}
// tsconfig.json
{
  ...
  "compilerOptions": {
    ...
    "baseUrl": ".",
    "paths": {
      "~/*": ["./src/*"]
    }
  }
}
```

The `~` alias is a preference. You can use other aliases such as `@` if you want.

### Add `cn` Helper Function

```ts
// src/lib/utils.ts
import { clsx, type ClassValue } from "clsx"
import { twMerge } from "tailwind-merge"

export function cn(...inputs: ClassValue[]) {
  return twMerge(clsx(inputs))
}
```
